## Description

The SHA256 Module for the computer API!

## Functions

### encode

```lua
sc.sha256.encode( str )
```

Encodes a string to be SHA256

**Arguments:**
- str [ **string** ] The string to convert to SHA256

**Returns:**
- [ **string** ] The converted string
